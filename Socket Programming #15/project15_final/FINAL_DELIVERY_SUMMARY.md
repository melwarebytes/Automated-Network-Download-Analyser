## 🎨 **VISUALIZATION SHOWCASE**

### **What the Charts Show:**

**Chart 1: Speed Over Time (Line Graph)**
```
Shows how download speed varies throughout the measurement period.
- X-axis: Time (hours:minutes)
- Y-axis: Speed (Mbps)
- Red dashed line: Average speed
- Blue solid line: Actual measurements
```

**Chart 2: Hourly Averages (Bar Chart)**
```
Average download speed for each hour of the day.
- Green bars: Above-average performance
- Red bars: Below-average performance
- Values labeled on each bar
- Identifies congestion periods
```

**Chart 3: Speed Distribution (Histogram)**
```
Frequency distribution showing speed consistency.
- X-axis: Speed ranges (Mbps)
- Y-axis: Number of downloads
- Green: Above mean
- Red: Below mean
- Shows if performance is consistent or variable
```

**Chart 4: Hourly Ranking (Horizontal Bars)**
```
Hours sorted by performance (best to worst).
- Top: Best performing hours
- Bottom: Worst performing hours
- Clear visual for identifying optimal times
```
